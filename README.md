# SafariSphere Tours

A modern, responsive web application for booking safari tours and charter services in Kenya.

## Features

- Interactive tour and charter booking system
- Real-time form validation
- Responsive design for all devices
- Modern UI with smooth animations
- Contact form with inquiry system

## Technologies Used

- React 18.3.1
- React Router DOM 7.9.5
- CSS3 with Modern Features
- Responsive Design
- Progressive Web App Support

## Getting Started

### Prerequisites

- Node.js 14.0.0 or later
- npm 6.0.0 or later

### Installation

1. Clone the repository
```bash
git clone git@github.com:Ibrahimgithinji/SafariSphere-Tours.git
```

2. Navigate to the project directory
```bash
cd safarisphere-tours
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm start
```

The application will open in your default browser at `http://localhost:3000`.

### Building for Production

To create a production build:

```bash
npm run build
```

This will create an optimized build in the `build` folder.
